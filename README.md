# 📋 AI 브랜드 광고 제작 미션 요약

과제 문서를 핵심만 정리해드릴게요!

---

## 🎯 미션 한 줄 요약
가상/실제 브랜드 1개를 정하고, **AI 도구만으로 10초 이내 광고 영상**을 만드는 과제입니다.
(도구를 많이 쓰는 게 아니라 **기획 → 프롬프트 → 결과물 → 편집** 파이프라인 설계가 핵심!)

---

## 📦 최종 제출물 (2가지)

| 산출물 | 형식 | 핵심 요건 |
|--------|------|-----------|
| **① 스토리보드** | PDF 1개 | 브랜드 정의 + 씬별 필수 필드 + 사용 도구 |
| **② 광고 영상** | MP4 1개 | 10초 이내, 720p~1080p, AI 소스 위주 |

### 씬별 필수 필드 (모든 씬 공통)
- 씬 번호 / 길이(초)
- 목표 메시지(한 문장)
- 화면 구성(구도·피사체·배경·텍스트)
- 내레이션/카피
- 사용 도구 + 사용 목적
- **프롬프트 원문 + 결과 요약**
- 결과 파일명/링크

---

## ✅ 반드시 지켜야 할 기능 요구사항
1. **브랜드 정의** — 타겟 / 톤앤매너 / USP + 핵심 메시지 1문장
2. **스토리보드 작성** — 씬 단위 + 최소 1개 씬 "프롬프트 수정 전/후" 기록
3. **멀티모달 도구 사용** — 이미지 / 비디오 / 오디오 AI **각 1종 이상**
4. **영상 스펙** — 10초 이내 + 시각 + 청각 요소 필수 포함
5. **서사 구조** — 기승전결 or 명확한 메시지 전달
6. **브랜딩 장치** — 마지막 3~5초에 로고·브랜드명·슬로건·CTA 중 1개 이상

---

## ⚠️ 주요 제약 사항
- ❌ 직접 촬영·유료 스톡 사용 금지 → **모든 소스는 AI 생성**
- ❌ 딥페이크·유해 콘텐츠 금지
- 편집 툴(Premiere, CapCut 등)은 **컷 편집·자막·색보정 정도만** 허용
- 💰 크레딧 절약: **이미지로 스토리보드 확정 후** 영상 변환
- 🔄 도구 대기열 대비 → 이미지/비디오/오디오 **대체 도구 각 1개** 준비

---

## 🌟 보너스 (선택)
1. **립싱크** — 인물 발화 장면 입모양 맞추기
2. **다른 도구로 재제작** — 같은 씬을 다른 도구로
3. **화면 비율 버전** — 16:9 / 9:16 / 1:1 등

---

## 💡 추천 진행 순서
```
① 브랜드·메시지 정의
② 스토리보드 초안 (씬 나누기)
③ 이미지 생성 → 스타일 확정
④ 이미지 → 영상 변환 (크레딧 최소화)
⑤ 오디오 생성 (BGM/내레이션/효과음)
⑥ 통합 편집 → MP4 완성
```
